HTML é uma linguagem de marcação utilizada
na construção de páginas na Web.
Documentos HTML podem ser interpretados por
navegadores.
A tecnologia é fruto da junção entre os padrões
HyTime e SGML.

JavaScript é uma linguagem de programação
estruturada, de script em alto nível com
tipagem dinâmica fraca e multiparadigma.
Juntamente com HTML e CSS, o JavaScript é
uma das três principais tecnologias
da World Wide Web.

CSS é um mecanismo para adicionar estilo a
um documento web.
O código CSS pode ser aplicado diretamente
nas tags ou ficar
contido dentro das tags < style >.
